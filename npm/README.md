# Rampyaaryan — India's First Hinglish Programming Language

Write code in the language you think in. No Python. No Node.js. No dependencies. Just one native binary.

## Install

```bash
npm install -g rampyaaryan
```

## Usage

```bash
# Run a file
rampyaaryan hello.ram

# Start interactive REPL
rampyaaryan

# Switch terminal to Hindi mode (UTF-8 + Hindi prompt)
rampyaaryan hindi

# Switch back to English
rampyaaryan hindi --off
```

## Example

```
likho "Namaste Duniya!"

maano naam = pucho("Aapka naam: ")
likho "Swagat hai, " + naam + " ji!"
```
